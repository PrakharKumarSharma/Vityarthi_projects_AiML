# Spam Email Classifier

##  Overview
This project classifies messages as Spam or Not Spam using Machine Learning.

##  Dataset
Due to large file size, full dataset is not included.

Download from:
https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset

After downloading:
- Rename file to dataset.csv
- Place it in project folder

##  Requirements
pip install -r requirements.txt

##  How to Run
python main.py

##  Example
Input: "Win money now!!!"
Output: Spam

##  Model Used
Naive Bayes

##  Accuracy
~95%
